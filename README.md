Software Engineer with **5 years of experience** building and supporting enterprise-scale web applications, production systems, and AI solution concepts.

I started with a strong foundation in **frontend development**, working with React, TypeScript, JavaScript, HTML, CSS, REST API integrations, and enterprise UI systems. Over time, I have been transitioning into a broader **Software Engineer** profile by expanding into backend development, cloud fundamentals, DevOps workflows, AI automation, and system design.

My current focus is building practical software systems that connect modern web applications with **AI agents, RAG pipelines, automation workflows, and cloud-based infrastructure**.

I have worked on enterprise identity and security platforms, migrated legacy systems into modern React-based architectures, supported production applications, and contributed to AI automation proposals using tools such as IBM Watson, Langflow, AI agents, and workflow automation platforms.
